NEOVIM 体验

本文最后更新于 2025-05-24T20:38:45+08:00

具体内容以后再补，先写快捷键的。

快捷键归纳

算是先前笔记的部分内容的重置。

模式切换

快捷键功能
i进入插入模式（光标前）
a进入插入模式（光标后）
o在下方新行插入
Esc / Ctrl+[返回普通模式
v进入可视模式（字符选择）
V行选择模式
Ctrl+v块选择模式

光标移动

快捷键功能
h j k l左/下/上/右移动
w / b跳到下一个/上一个单词开头
e跳到单词末尾
0 / $行首/行尾
gg / G文件开头/结尾
Ctrl+d / Ctrl+u向下/上翻半页

文本编辑

快捷键功能
x删除当前字符
dd删除当前行
yy复制当前行
p / P粘贴到光标后/前
u / Ctrl+r撤销/重做
.重复上一次操作

窗口和标签页

快捷键功能
Ctrl+w s水平分割窗口
Ctrl+w v垂直分割窗口
Ctrl+w h/j/k/l切换窗口方向
:tabnew新建标签页
gt / gT下一个/上一个标签页

搜索与替换

快捷键功能
/关键词Enter向下搜索
?关键词Enter向上搜索
n / N跳转到下一个/上一个匹配
:%s/old/new/g全局替换

其他实用功能

快捷键功能
Ctrl+z挂起 Neovim ，返回终端
fg恢复挂起的任务
:!命令Enter执行外部命令（如 :!python % 运行当前文件
